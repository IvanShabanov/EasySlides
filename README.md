# EasySlides
Jquery plagin slides

Usage

   
   
      $('.slider').EasySlides({
            'autoplay': true, 
            'timeout': 3000,
            'show': 5, //how many items not hidden
            'vertical': false,  //is vertical slider. If true slides changing by up/down mouse move
            'reverse': false, //Reverse slide–∫ control 
            'mouseevents': true,
            'beforeshow': function () {},
            'aftershow': function () {},      
      });
